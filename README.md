# EduTrack JS

EduTrack JS is a classroom management web application built using HTML, CSS, and JavaScript.

## Features
- Add and manage students
- Add grades
- Automatic average calculation
- Data stored in localStorage
- Dynamic table updates

## Technologies Used
- HTML
- CSS
- JavaScript (DOM manipulation, events, localStorage)

## Purpose
This project demonstrates advanced JavaScript skills and building a dynamic user interface without frameworks.
